# How to Follow the Roadmap

This repository is designed to be followed **sequentially**.

### Recommended Order

1. Start with `track-00-angular-13`
2. Complete each section in the given order
3. Do not skip foundational topics
4. Move to the next Angular version only after understanding the previous one

### Learning Approach

- Each folder represents a **logical learning stage**
- Each `.md` file focuses on **one concept only**
- Take notes, experiment with code, and revisit files when needed

### Important Note

This is **not a crash course**.

The goal is to:
- Build correct mental models
- Understand architectural decisions
- Learn Angular in a way that scales with experience

Consistency matters more than speed.
