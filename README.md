# angular-beginner-to-intermediate

A structured, step-by-step Angular learning repository covering **beginner to intermediate concepts** with real examples.

---

## 🚀 What This Repository Covers

- Learned all **core Angular concepts using Angular 13**
- Implemented concepts with **practical and structured examples**
- Progressively **migrated the project from Angular 13 → 18 (version by version)**
- Explored **new features introduced in each Angular release**
- Documented **deprecated features and modern alternatives**
- Built a strong foundation in:
  - Components & Lifecycle
  - Data Binding
  - Directives & Pipes
  - Dependency Injection
  - Routing
  - Forms
  - HTTP & API Handling
  - RxJS
